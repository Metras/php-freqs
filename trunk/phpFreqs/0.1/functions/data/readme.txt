Formatting:

This directory will be read upon intial install to load locale information into the database.

Main Folders will be read as Country information.

Files inside folders will be read as states/provinces.

Content inside the files will be read as cities/towns.

**** Please note.
Each location in the files should be entered one per line,
Please re-sort the file after adding information.
Please also add a <cr> after the last location entered.
This ensures the EOF will happen after the last location is read.
